# 📖 株ボールシステム マニュアル

## 1. はじめに
株ボールシステムは、LINE Bot + ダッシュボードで投資を支援する AI エージェントです。
- 📱 LINE Bot: 売買記録・AI相談・指値アラート
- 📊 ダッシュボード: グループ内ランキング・タイムライン・セクター分析
- 🤖 AI解説: 敏腕トレーナーが株価の動きを3行で解説

## 2. ダッシュボードの使い方
https://stock-ai-agent-ball.streamlit.app/?code=<招待コード>

招待コードは LINEグループで /register したときに Bot が発行します（例: M3DFIU）。

- 🏆 ランキング: グループメンバーの損益ランキング
- 📜 タイムライン: 売買・コメントの時系列
- 📊 セクター集計: セクター別勝率
- 🚨 アラート履歴: 重大アラート一覧

## 3. 個人LINEの使い方
株ボールシステムBotを友だち追加したら、普段人間と話すように自然な言葉でチャットするだけでOKです！
難しいコマンドを覚える必要はありません。AIがあなたの言葉の意図を読み取って自動で処理します。

### できることの例（そのまま話しかけてください）
- **監視・指値アラート**: 「NVDAを監視リストに入れて」「NVDAが140ドルになったら教えて」
- **売買の記録**: 「今日AAPLを150ドルで10株買ったよ」「TSLAを全部売却した」
- **AI分析・相談**: 「半導体セクターの今後の見通しはどう？」「NVDAのテクニカル分析をお願い」
- **ポートフォリオ確認**: 「今の保有状況を教えて」「AIにポートフォリオの改善点を相談したい」
- **目標・成績管理**: 「今月の進捗はどう？」「今月の損益グラフを出して」

> [!IMPORTANT]
> **情報のソース（出典）について**
> 株は情報が命です。AIが銘柄分析やニュースを提供する際は、**必ずリンクや参考元（ソース）を提示**するように設計されています。もし情報元がわからない場合は「その情報のソースを教えて」と遠慮なく聞いてください。

*(※従来通りの `/add NVDA 140` や `/buy AAPL 150 10` のようなコマンド形式でも引き続き利用可能です。`/help` で一覧が見られます)*

## 4. グループLINEの使い方
Bot をグループに招待すると、以下の「大きなお知らせ」系コマンドが使えます。

- `/register <グループ名>` — このLINEグループをシステムに登録（初回必須）
- `/ranking [日数]` — グループ損益ランキング（例: /ranking 30）

個別の売買共有・コメント・分析は**個人チャット**で行ってください。

## 5. 指値アラートの設定方法

### 設定する
個人LINEで:
```
/add NVDA 140
```
→ NVDA が 140 USD に接近したら通知

### 変動率アラート
デフォルトで前日比 ±5% を超えたら通知。閾値変更は管理者に依頼してください。

### 解除する
現在、解除コマンドは未実装。管理者に連絡してください。

## 6. よくある質問

### Q. Bot が反応しない
- 友だち追加していますか？ QRコードで追加してください。
- メッセージが「自動応答」で返される場合、Bot側の「応答設定」で Webhook が ON になっていません。

### Q. 招待コードがわからない
- LINEグループで `/register <グループ名>` と送ると、Bot が招待コードを発行します。
- もしくはグループ管理者に聞いてください。

### Q. ダッシュボードが表示されない
- 招待コードが正しいか確認してください。URL例: `?code=DEMO01`
- 一時的なメンテナンス中の可能性あり。時間を置いて再試行してください。
